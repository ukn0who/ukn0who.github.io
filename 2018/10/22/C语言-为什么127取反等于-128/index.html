Q&A / C 语言- 为什么127取反等于-128?

在求signed 和 unsigned 类型范围的文章中, 通过位计算的方式, 算出了类型的范围. 其中的一个让我比较疑惑的点是, 范围的最大值取反, 就是范围的最小值.

取反, 这个操作简单来讲就是将 1 置为 0 , 将 0 置为 1. 在有符号的类型中,

​ 127 = [ 0111 1111 ]

​ ~127 = [ 1000 0000 ]

为什么 ~127 不等于 -0 呢?
